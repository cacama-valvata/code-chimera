def getFromDict(dataDict, mapList):    
    for k in mapList: dataDict = dataDict[k]

And the code from @eafit's answer for setting a value:

def nested_set(dic, keys, value):
    for key in keys[:-1]:
        dic = dic.setdefault(key, {})
    dic[keys[-1]] = value
